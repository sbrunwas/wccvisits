# Well-Child Vaccine Timing (CA 2025)

Small Streamlit prototype that looks up routine well-child vaccine timing visit buckets based on California CDPH IMM-395 (2025) schedule values.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes

- Uses a hard-coded lookup table (no external APIs, no database).
- Intended for prototype/educational use only, not medical advice.
